# 🔐 Scénario simple

## Situation
Un employé rejoint une entreprise.

## Étapes
1. Création de son compte utilisateur
2. Ajout à un groupe (ex : service informatique)
3. Attribution de droits
4. Connexion au poste

## Résultat
L’utilisateur accède uniquement aux ressources autorisées.

## Conclusion
Active Directory permet de gérer facilement les accès.
